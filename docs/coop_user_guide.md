# Co-op（2 人プレイ）ガイド

1 つの画面で **2 人同時プレイ**を楽しむための機能です。あなた（P1）の隣に相方（P2）を呼び出し、
**2 本目のコントローラ**で操作できます。

> ⚠️ **実験的な機能（WIP）です。** 一部のステージ・組み合わせで挙動が不安定なことがあります。
> オフラインの練習・お遊び用としてお使いください。

---

## 用意するもの

- **コントローラ 2 本**（P2 用に 2 本目を接続しておく）
  - P1 はいつもの操作（キーボード / 1 本目のコントローラ）
  - P2 は **2 本目のコントローラ**で動かします

> 💡 **離れた友達とオンラインで遊ぶには**: [Parsec](https://parsec.app/) などのリモートプレイアプリを
> 使えば、相手のコントローラがこの PC の「2 本目のコントローラ」として認識され、
> オンラインでも一緒に遊べます（遅延は回線環境に依存します）。

---

## 使い方

1. ミッション中に trainer を開く（**INS** か **F1**）
2. **Game タブ → 「Co-op」** を開く
3. **「Co-op (P2)」にチェック**を入れる
  - これだけで カメラ・2 本目コントローラ・ロックオンが 2 人用にまとめて切り替わります
4. **P1（操作キャラ）** と **P2（相方）** を選ぶ
  - **P1**：あなたが操作するキャラを切り替え
  - **P2**：隣に呼び出す相方のキャラを選ぶ（**P1 と違うキャラでも OK**）
  - P2 を「(なし)」にすると相方を引っ込めます
5. 2 本目のコントローラで相方を動かす

> P1 / P2 の選択は **「Co-op (P2)」が ON のときだけ**操作できます（OFF のときはグレー表示）。

---

## できること

- **別キャラの相方を呼べる** — 例：ダンテ＋ネロ、バージル＋トリッシュ など自由な組み合わせ。
- **それぞれ自分のスキルで戦える** — 相方は、そのキャラ本来の解放済みスキルで動きます。
- **カメラが 2 人を自動で収める** — 2 人（とお互いのロックオン相手）が画面に収まるように、
カメラの位置と引き具合が自動調整されます。**ボス戦もカメラが 2 人を追います**。
- **ロックオンが別々** — P1 と P2 は、それぞれ **自分の近くの敵**にロックオンします。
- **倒れても自動で復活** — 相方が生きていれば、ダウンから数秒で自動蘇生します（下記参照）。
- **相方のステータスが画面に出る** — 相方の HP・ゲージ・武器を、画面に自動で表示します（Co-op ON で自動）。
- **敵が 2 人に散らばる** — 敵は近い方のプレイヤーを狙うようになります。
- **相方の技もちゃんと出る** — ネロのスナッチ／バスターの位置や投げる向きが、相方側でも正しく決まります。
- **ステージを移動しても相方が残る** — ステージが切り替わっても、選んでいた相方が自動でまた出てきます
（ボスステージでも自動で出てきます）。
- **対戦もできる** — Friendly Fire / PvP を ON にすれば 2 人で撃ち合い・斬り合いができます（下記参照）。

---

## 相方（P2）のロックオン操作

P2 は非ロックオン時、**最寄りの敵を自動で狙います**。手動ロックオンも使えます：

- **ロック維持（既定 R1）** — 押している間、ターゲットを固定します。離すと自動ターゲットに戻ります。
- **ターゲット切替（既定 L3 = 左スティック押し込み）** — ロック維持中に押すと次の対象へ切り替え。
- ボタンの割り当ては **「P2 ロックオン設定」** で変更できます。

---

## ダウンと自動蘇生

- 片方が倒れても、**相方が生きていればゲームオーバーになりません**。ダウンして数秒待つと自動で復活します。
- 蘇生までの残り時間は画面に **カウントダウン表示**されます（P1 は左下・P2 は右下）。
- **「ダウン秒数」「蘇生 HP」** スライダーで、待ち時間と復活時の HP を調整できます。
- **2 人同時に倒れると通常どおりゲームオーバー**。

---

## Friendly Fire / PvP（対戦）

- **Friendly Fire** — P1/P2 のお互いの攻撃が当たるようになります。
- **PvP**（Friendly Fire ON 中のみ）— お互いを**ロックオン**でき、**エネミーステップ**（相手踏み）も可能に。
タイマン対戦用です。PvP 中の蘇生は常に HP 全快です。
- **攻撃力補正** —敵へ与えるダメージの倍率を調整できます
（PvP の対人ダメージには影響しません）。

---

## 相方（P2）のワープ

グリムグリップなどのギミック移動で相方が置き去りになったら、**「P2 を P1 の位置へ」ボタン**
（または既定 **HOME** キー、変更可）で P1 の隣へテレポートさせられます。

---

## カメラの調整

「Co-op」内の **「カメラ調整」** を開くと、見やすさを好みに合わせられます。


| 項目                   | 説明                               |
| -------------------- | -------------------------------- |
| 引きの強さ                | 2 人（や敵）が離れるほどカメラを引く量。大きいほどよく引きます |
| 最小 / 最大ズーム           | 引きの下限 / 上限                       |
| （通常時 / ロックオン中 / ボス戦） | それぞれ別に調整できます                     |
| 追従の滑らかさ              | カメラの動きの滑らかさ。小さいほどゆっくり追従          |
| 固定カメラの追従倍率           | 固定アングルのカメラで 2 人へ寄せる量。動きすぎるなら下げる  |
| **リセット**             | カメラ調整の値をすべて初期値に戻す                |


迷ったら **「リセット」** で初期値に戻せます。

---

## 相方（P2）の HUD 位置

「Co-op」内の **「P2 HUD」** を開くと、相方のステータス表示の位置を動かせます。

- Co-op を ON にすると相方の HUD は**自動で表示**されます（個別の ON/OFF は不要）。
- **X 位置 / Y 位置** で表示場所を調整できます。画面サイズに対する **%（相対）指定**にも切り替えられるので、
解像度が変わっても崩れにくくできます。

---

## うまくいかないときは

- **相方が出てこない / 動かない**
  - 「Co-op (P2)」が ON か、P2 のキャラが選ばれているか確認
  - 2 本目のコントローラが認識されているか確認
- **直後にクラッシュする / 不安定**
  - セーブデータをロードしてから使う（リソースの読み込みのため）
  - まずはトレーニングモードなど安全な場所で試す
- **ステージ移動の直後は**、相方が出るまで一瞬（約 0.5 秒）かかります

---

## 注意点・既知の制限

- 相方は **1 人まで**（コントローラ 2 本での 2 人プレイ）。
- **ボス（クレド以外）は P1 を優先して狙います**。
- ロックオンの **カーソル（照準マーク）は P1 側のみ**表示されることがあります
（相方の攻撃の狙い自体は、相方の対象に向きます）。
- 相方の HUD レイアウトはキャラによって調整途中の場合があります。
- 一部の組み合わせ・場所で挙動が乱れることがあります。
